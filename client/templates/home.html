Tutorial

MAKE GITHUB ACCOUNT


Head over to github.com
and make an account if you don't have one already.

FORK CODE


Go to the Biomodd London Repository
and press "fork" after you logged in.

GET GUI 4 GITHUB


If you know your away around git, please use your go-to system. Otherwise download a GUI/GIT client for your development OS.
Windows (official)
Mac (official)
Linux gui
Linux Commandline
After installing, log in with your Github account and clone your fork of the "Biomodd London Repository" on your machine.

RUN THE SKETCH LOCALLY

On UNIX (osx/linux)

  1. Open Terminal
  2. type [cd "PATH"] where PATH should be replaced with the path to your clone of the repository, for example; cd ~/Desktop/myBiomoddClone
  3. type [python -m SimpleHTTPServer]
  4. open op your favorite browser and browse to "http://localhost:8000/"

On Windows

  1. Open MS-DOS (type "cmd" in "start")
  2. type [cd "PATH"] where PATH should be replaced with the path to your clone of the repository, for example; cd ~/Desktop/myBiomoddClone
  3. type [python -m http.server 8000]
  4. open op your favorite browser and browse to "http://localhost:8000/"

You should see a fullscreen sketch running right now, use the arrow keys to controll your avatar.

START MODDING

Open up your local clone of the repository using your favorite text editor. If you are in doubt, you could use sublime text
Open up "code.js" in the text-editor, and try changing some of the variables
//---------------------------------- // variables //---------------------------------- /** * Background: The background color for our sketch, can be modified * @type {String} * @namespace changeable variables */ this.backgroundColor = '#660066';

In general most of those lines should be visible in grey if you open them up in a text-editor. This means that they are comments, and not really code that executes into software. The only meaningful line for us is
this.backgroundColor = '#660066'; //rgb(102, 0, 102)

Try changing the value '#660066' into something different, safe in the text-editor and reload "http://localhost:8000/" in your browser. (the #660066 represents a hexadecimal color)

Congratz: you did your first game-modding.
